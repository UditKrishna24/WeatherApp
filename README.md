# 🌦️ Weather App

A native Android Weather Application developed using **Java** and **Android Studio** that provides real-time weather information for cities worldwide using a REST API.

The application allows users to search for any city and displays current weather conditions including temperature, humidity, wind speed, pressure, visibility, and weather description through a clean and responsive interface.

---

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** Android Studio
- **UI:** XML Layouts
- **Networking:** Retrofit
- **HTTP Client:** OkHttp
- **JSON Parsing:** Gson
- **View Binding**
- **REST API Integration**
- **Gradle Build System**

# 📥 Installation

Download the latest APK from the **Releases** section of this repository.

Install the APK on an Android device running Android 7.0 (Nougat) or above.

If prompted, allow installation from unknown sources.
